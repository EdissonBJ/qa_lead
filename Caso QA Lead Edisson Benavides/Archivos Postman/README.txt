# Greentex – QA Lead Technical Test

## Cómo ejecutar la colección

1. Importar el archivo `stageGreentex.postman_collection.json`.
2. Importar el environment: `Greentex-Stage.postman_environment.json`.
3. Seleccionar el environment en Postman.
4. Ejecutar la colección completa usando "Run Collection".
5. Verificar:
   - Status codes
   - Mensajes de validación
   - Tiempos de respuesta
   - Resultados pasados/fallidos

## Estructura
- Formulario de contacto (valid/invalid)
- Lista de productos (valid/invalid)


## Evidencia
Incluye captura de:
